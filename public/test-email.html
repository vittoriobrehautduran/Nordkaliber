Setup Instructions

1. Gmail App Password: You need to create an app password for nordkaliber@gmail.com

2. Environment Variables: Set these in Vercel:

  • EMAIL_USER=nordkaliber@gmail.com
  • EMAIL_PASSWORD=your_app_password
  • PRODUCTION_MANAGER_EMAIL=nordkaliber@gmail.com

3. Gmail Settings: Enable 2-factor authentication and generate an app password
